# 📮English to Gujarati Mailbox
Translating my grandma's mailbox so she can listen to her emails. 

### 📋Aims of App

- Connect app to gmail/hotmail
- Enable user to view emails (with translations in English and Gujarati)
- Click on the email to view it's title, date and contents with the option to listen to it too 

### 👩‍💻Technical Details

- React
- TypeScript
- Next.js
